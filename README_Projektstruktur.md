# Projektstruktur – Predictive Maintenance Deutsche Bahn

Dieses Projekt besteht aus mehreren Modulen zur Analyse, Modellierung und Präsentation eines Predictive-Maintenance-Systems für ICE-Komponenten.

## Ordnerübersicht

```
PredictiveMaintenance_DB/
│
├── 01_Hausarbeit/
├── 02_Daten/
├── 03_Notebooks/
│   └── Ergebnisse/
├── 04_Code/
├── 05_Praesentation/
└── 06_Dokumentation/
```

### 01_Hausarbeit/
Beinhaltet die schriftliche Fallstudie (PDF/DOCX) sowie Literatur.

### 02_Daten/
Beinhaltet den synthetischen Datensatz `predictive_maintenance_db.csv` und Datenbeschreibungen.

### 03_Notebooks/
Alle Jupyter-Notebooks zur Modellierung. Unterordner `Ergebnisse` enthält generierte Grafiken.

### 04_Code/
Python-Skripte zur Datengenerierung und Modellierung:
- `datengenerierung.py`
- `train_model.py`

### 05_Praesentation/
Folien und Grafiken für Präsentationen.

### 06_Dokumentation/
Wichtige Zusatzdokumente (Modellbeschreibung, Prüfungsnotizen).
